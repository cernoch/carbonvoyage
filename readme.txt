Wordpress plugin for calculating journey's carbon footprint

== Installation ==

1. Install the plugin into `/wp-content/carbonjourney` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place `[carbon]Destination[/carbon]` text in your document.
